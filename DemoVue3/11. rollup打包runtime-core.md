## 空包构建
空包构建指不考虑最终产物的情况下跑通整个构建流程，最终产物可以是一个无代码的js文件。
空包构建成功，意味着流程初步跑通。
#### runtime-core 统一导出
1. 在runtime-core和src文件夹下创建index.ts文件
```ts
// runtimer-core/index.ts
export { createApp } from './createApp';
export { h } from './h';

// src/index.ts
export * from './runtime-core/index'
```
